## 📚 Overview

This repository contains **Week 4 classroom demonstrations and practical examples** focused on:

> White-Box Functional Testing using C# console applications

Students analyze **internal program logic**, verify **decision structures**, and measure **test coverage quality** using structured testing techniques aligned with professional software testing practices.

White-box testing enables testers to inspect **how the program works internally**, not only what it produces externally.

---

## 🎯 Learning Outcomes

By completing this practical session students can:

✔ Perform static testing on source code
✔ Trace execution paths inside programs
✔ Test conditional logic structures
✔ Measure different types of code coverage
✔ Apply error-guessing strategies
✔ Test modules and functions independently
✔ Understand Gray-Box testing fundamentals

✔ invalid input

---

### 📊 Code Coverage Techniques

Coverage measured during practical exercises:

| Coverage Type      | Purpose                 |
| ------------------ | ----------------------- |
| Function Coverage  | Execute every function  |
| Statement Coverage | Execute every line      |
| Decision Coverage  | Execute all branches    |
| Condition Coverage | Evaluate variables      |
| Loop Coverage      | Validate repetitions    |
| Parameter Coverage | Test input combinations |
